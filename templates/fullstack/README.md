# {{projectName}}

Full-stack MERN application with Express backend and React frontend.

## Getting Started

```bash
npm run install:all
npm run dev
```

Backend runs on http://localhost:3000
Frontend runs on http://localhost:3001

## Features

### Backend
- Express.js REST API
- MongoDB with Mongoose
- JWT Authentication
- Request validation

### Frontend
- React with Vite
- Tailwind CSS
- React Router
- Axios for API calls

## Project Structure

```
backend/          # Express API
frontend/         # React application
```
